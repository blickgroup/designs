Layer 2

AI Inference.
Mostly free.
Always private.

Three models run locally on the Mac Mini. Cloud is used only when local cannot serve the task. Drive file content never touches a cloud LLM — the nexus-drive gateway fails closed if Ollama is unavailable.

The LLM router (llm-router.js) makes every routing decision. Scan/filter/classify → e4b. Draft/synthesis → 26b. Architecture/strategy → Sonnet/Opus.

Resilience

Three layers.
Detects. Escalates. Fixes.

The self-healing stack is a 3-layer pipeline. Layer 1 detects and auto-resolves simple issues. Layer 2 promotes complex issues to the Action Register. Layer 3 drafts code fixes.

Real wins: 56-day silent failure · 98% duplicate inventory · 17 GB n8n disk-fill · U+FFFD burst · Compass sort bug caught by AI.

1
Night Watchman
health-collector.js · FAST every 30 min (7am–7pm) + DAILY 08:15
Watches 9 sources. Auto-resolves n8n workflow deactivation and missed scheduled runs. 18h cooldown per issue prevents alert fatigue. State: nexus/data/health-collector-state.json
2
Candidate Extractor
candidate-extractor.js · Triggered at end of daily health-collector
Promotes warning/critical findings to the Action Register as structured action items. Deduplication via entity fingerprints. Output: action_register entries with domain: infrastructure.
3
Fix Drafter
fix-drafter.js · Triggered manually on action register items
Gemma 26b drafts minimal code fixes for Sentry/CI-shaped action register items. Confidence-gated. Option B (LIVE): Writes draft fix to working copy for human review. Option A: Auto-PR (deferred).
